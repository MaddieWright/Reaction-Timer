# Reaction-Timer

### Function

- Tests human reaction time using lights or sound and records the fastest/average response.

### Outline

- Interface: LED or buzzer starts signal; user presses a button when they respond. MSP430 calculates reaction time and displays it on LCD.
- Hardware: MSP430, pushbuttons, LEDs or buzzer, small LCD display.

### Parts

- **MSP430** - central processor
- **Push button** - user responds to stimulus
- **LEDs** - indicate stimulus or speed
- **LCD** (optional) - numeric readout and scores
- **Buzzer** (optional) - auditory feedback
- **Resistors, wires, breadboard**
